# HospitalityClaw

Hotel and hospitality management for [ERPClaw](https://github.com/avansaber/erpclaw). Rooms, reservations, front desk operations, housekeeping, guest profiles, revenue management, and F&B outlets. 73 actions across 8 domains. Operator-side hotel management -- guest-facing booking is handled by third-party channels.

## Features

- **Rooms** -- room types, room inventory, amenities, availability reports
- **Reservations** -- bookings, rate plans, group blocks, confirmation and cancellation
- **Front Desk** -- check-in/check-out, room assignments, folio charges, guest requests, late checkout, room moves
- **Housekeeping** -- task scheduling, inspections, dashboard, laundry summaries
- **Guests** -- guest profiles, preferences, VIP levels, stay history, loyalty summaries
- **Revenue Management** -- rate adjustments, seasonal pricing, occupancy forecasts, RevPAR, ADR, yield analysis
- **Food & Beverage** -- outlets, room service orders, minibar consumption tracking
- **Reports** -- occupancy, revenue, housekeeping, guest satisfaction, daily operations, department performance

## Installation

Requires [ERPClaw](https://github.com/avansaber/erpclaw) core.

```
install-module hospitalityclaw
```

## Quick Start

```
"I'm running a hotel -- set me up"
```

The bot will install HospitalityClaw and guide you through setting up room types, managing reservations, and tracking revenue.
